🚨 시스템 연동 안내: 파이어베이스 보안 규칙 업데이트 필요

이 페이지가 정상적으로 작동하려면 데이터베이스에 3층 로케이션을 위한 저장 공간 권한을 추가해야 합니다. 파이어베이스의 [Firestore Database] -> [Rules (규칙)] 탭에 아래 코드를 추가해 주세요.

match /location_map_3f/{document=**} {
  allow read, write: if true; 
}

※ 코드를 추가하고 [게시(Publish)]한 뒤, 이 페이지를 새로고침(Ctrl + Shift + R)하면 이 안내문은 자동으로 사라지고 시스템이 정상 가동됩니다.
